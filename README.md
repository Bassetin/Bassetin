## 🕹️ Sobre Mim

Olá! Sou Lucas, estudante de Ciências da Computação no 6º período e desenvolvedor com experiência na construção de aplicações e sistemas, além de desenvolvimento de jogos, utilizando Java, Python e C#.

Tenho interesse em desenvolvimento de software e construção de aplicações interativas, com forte base em lógica de programação e Programação Orientada a Objetos. Já desenvolvi projetos completos, incluindo um jogo 2D em Java e uma aplicação em Python com integração a APIs e sistema de recomendação.

Também possuo experiência como professor de inglês, informática e programação, o que me proporcionou uma boa comunicação e entendimento de como as pessoas interagem com sistemas. Além disso, servi no Exército Brasileiro, onde desenvolvi disciplina, foco sob pressão e responsabilidade.
